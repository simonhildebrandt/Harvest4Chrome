h1. Harvest4Chrome

An extension designed to make keeping your Harvest timers even easier.

h2. Instructions

# Download and install "extension":https://github.com/simonhildebrandt/Harvest4Chrome/raw/master/HarvestExtension.crx
# Set your username and password in the extension options
# Watch the extension badge (Harvest icon in the corner) update with your current timer.
# Click badge to see all your timers for the day (or to see the current error, probably caused by mistyped credentials)
# Switch timers by clicking on them
# Watch for the 'angry' Harvest icon - you have no timer currently running!
